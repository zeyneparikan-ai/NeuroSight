# 🧠 NeuroSight

Görme engelli bireyler için geliştirilmiş gerçek zamanlı nesne tanıma ve sesli uyarı sistemi.

## 🚀 Özellikler
- Kamera ile gerçek zamanlı nesne tespiti (YOLOv8)
- Türkçe sesli uyarı sistemi
- Nesnenin yönünü ve mesafesini algılama
- Tehlike analizi

## 🛠️ Kullanılan Teknolojiler
- Python
- YOLOv8 (Ultralytics)
- OpenCV
- gTTS (Google Text-to-Speech)

## ▶️ Çalıştırmak İçin
pip install ultralytics opencv-python gtts playsound
python app.py
